### Deployment Options

**1. Vercel** (Recommended)
```bash
npm install -g vercel
vercel --prod
```
⏱️ 2 minutes | 🎯 Auto-scaling | 💰 Free tier available

**2. Netlify**
```bash
npm run build
# Drag dist/ to Netlify
```
⏱️ 5 minutes | 🎯 Easy setup | 💰 Free tier available

**3. GitHub Pages**
```bash
npm run deploy  # (with gh-pages configured)
```
⏱️ 10 minutes | 🎯 Version control | 💰 Free

**4. Docker**
```bash
docker build -t sanjeevini .
docker run -p 80:80 sanjeevini
```
⏱️ 15 minutes | 🎯 Full control | 💰 Any host

**5. AWS S3 + CloudFront**
```bash
npm run build
aws s3 sync dist/ s3://bucket/
```
⏱️ 20 minutes | 🎯 Global CDN | 💰 Scalable
